# Keliling UI - Backend

This is the backend server for the Keliling UI Monopoly game, built with Rust and Axum.

## Getting Started

### Prerequisites
- Rust (latest stable)

### Running Development Server

```bash
cd backend
cargo run
```

The server will start on `127.0.0.1:3000`.

## Project Structure

See `../technical_documents.md` for the detailed architecture and file structure.

## Tech Stack
- Axum (Web Framework)
- Tokio (Async Runtime)
- Serde (Serialization)
- Dashmap (Concurrent State)
