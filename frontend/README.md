# Keliling UI - Frontend

This is the frontend for the Keliling UI Monopoly game, built with React, TypeScript, and Vite.

## Getting Started

### Prerequisites
- Node.js (latest LTS recommended)
- Bun (optional, but recommended for speed)

### Installation

```bash
cd frontend
bun install
# or
npm install
```

### Running Development Server

```bash
bun run dev
# or
npm run dev
```

The app will be available at `http://localhost:5173`.
